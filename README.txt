# Project Title

entertainment_center.py is a program that creates a simple html website that displays a number of movies posters which when clicked on displays an embedded YouTube video of the trailer.

## Getting Started

Download the zip file which contains all the required code to make the program run.

### Prerequisites

Have some version of Python installed or a compatible IDE

### Installing

Unzip files into chosen directory

Make sure all files are in the same directory

Open IDLE (Python GUI)

Open entertainment_center.py

Run entertainment_center.py or press F5 in IDLE window
